# Service Auth

Servicio para autorizacion a travez de validacion de tokens

## How to use

[Clona](https://github.com/hipotecariofacil/services-auth.git) el repositorio :

```bash
git clone https://github.com/hipotecariofacil/services-auth.git
```

Install it and run:

```bash
npm install
npm run dev
```

## Endpoints

- /auth/signup
- /auth/login
- /auth/validate
